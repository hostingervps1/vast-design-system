# 输入框组件

## 核心定位
唯一交互入口，用户通过自然中文语言驱动所有操作。

## 规格

```
宽度：100%（全宽）
高度：最小 56px，自动撑高（多行）
位置：固定底部
背景：rgba(26, 26, 38, 0.92) + backdrop-filter: blur(12px)
边框：无，顶部 1px rgba(108, 99, 255, 0.2)
圆角：16px 16px 0 0
```

## 占位文字
```
"用中文告诉我你想做什么..."
颜色：#44445A
字号：15px
```

## 发送按钮
```
形状：圆形 44px × 44px
背景：#6C63FF（有内容时）/ #1A1A26（空时）
图标：箭头向上
位置：右下角，间距 8px
```

## 状态
- **空**：按钮灰色禁用
- **输入中**：按钮激活，主色
- **发送中**：按钮转圈动画
- **流式输出中**：显示停止按钮

## 快捷功能栏（输入框上方）
```
[📎 附件] [🎤 语音] [🖼️ 图片] [⚡ 快捷指令]
高度：40px，横向滚动，间距 12px
```
