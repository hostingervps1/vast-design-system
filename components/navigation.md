# 导航组件

## 三模块切换

### 移动端（底部 Tab 栏）
```
高度：64px + 安全区
背景：#12121A，顶部 1px rgba(255,255,255,0.06)
三个 Tab 等分
```

| Tab | 图标 | 名称 |
|-----|------|------|
| 画图模块 | 🎨 | 创作 |
| N8N推理 | 🔄 | 推理 |
| 生活百科 | 📚 | 百科 |

**激活态：** 图标主色 #6C63FF + 标签加粗  
**非激活：** 图标 #44445A + 标签 #8888AA  
**切换动画：** 200ms ease

### 桌面端（左侧图标栏）
```
宽度：64px（收起）/ 220px（展开）
背景：#0A0A0F
切换：hover 或点击展开
```

## 顶部栏
```
高度：56px
左：VAST Logo（24px）+ 当前模块名
右：设置图标 + 用户头像
背景：#0A0A0F，底部 1px rgba(255,255,255,0.06)
```

## 会话历史侧边栏
```
宽度：280px，从左滑入
背景：#0A0A0F
条目高度：48px，悬停 #12121A
时间分组：今天 / 昨天 / 更早
```
