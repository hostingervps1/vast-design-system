# VAST AI 设计系统

> **VAST人工智能公司** — 以 Claude 为核心大脑，纯自然中文语言驱动的三端私人定制平台

---

## 公司定位

- **核心理念**：零代码，一律使用自然中文语言驱动
- **目标用户**：少数高端用户，私人定制，非大众化产品
- **三端覆盖**：APP / IDE / CLI
- **智能核心**：Claude 所有能力自动内置加载，无需用户手动配置

---

## 产品架构

### 三大功能模块（聊天版）

| 模块 | 功能 | 说明 |
|------|------|------|
| 🎨 画图模块 | AI 图像生成与编辑 | 自然语言描述即可生成图像 |
| 🔄 N8N 推理线路 | 自动化工作流推理 | 可视化逻辑链，AI 自动编排 |
| 📚 日常生活百科 | 生活知识问答助手 | 覆盖衣食住行全场景 |

### 内置能力（对所有模块统一）

- **操作终端能力**：每个模块内置类 Claude Code 终端，可执行系统级操作
- **统一网关**：无需客户主动连接，出厂预配置完毕
- **全自动初始化**：所有可自动配置的功能，一律内置，开箱即用

---

## 设计原则

### 1. 极简交互
- 唯一输入方式：自然中文语言
- 无学习成本，无配置步骤
- 界面元素只保留必要信息

### 2. 私人定制感
- 非标准化 UI，每个客户独立实例
- 深色高级质感为主基调
- 无广告、无多余功能入口

### 3. 智能优先
- AI 主动理解意图，而非被动响应指令
- 上下文持久记忆，无需重复说明
- 错误自动修复，流程自动推进

## 组件规范

### 输入框（核心组件）

- 全宽，底部对齐屏幕
- 无边框，半透明磨砂背景
- 占位文字：`用中文告诉我你想做什么...`
- 发送按钮：圆形，主色填充

### 消息气泡

- 用户消息：右对齐，主色边框
- AI 回复：左对齐，无边框，代码块深色高亮
- Markdown 完整渲染

### 模块切换导航

- 底部 Tab 栏（移动端）/ 左侧图标栏（桌面端）
- 当前模块图标亮色填充，其余灰色
- 切换动画：200ms ease

## 技术规范

### 三端统一网关

```
协议：WebSocket + HTTP/2
认证：设备指纹 + 预配置密钥（无需用户登录）
连接：出厂自动建立，断线自动重连
```

### Claude 核心集成

```
模型：claude-sonnet-4-6（默认） / claude-opus-4-7（高级任务）
上下文：持久化，跨会话记忆
工具：全量加载（代码执行 / 文件操作 / 网络访问 / 终端控制）
语言：纯中文交互，内部自动处理
```

### 部署架构

```
VPS（62.72.1.199）— 核心枢纽
  ├── Claude API 网关
  ├── N8N 工作流引擎
  ├── 用户会话管理
  └── 文件/数据存储

客户端（APP / IDE / CLI）
  └── 自动连接 VPS 网关，无需配置
```
